^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

1.1 访问路径

>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

=============== =============== =============================================
    userId          String                       当前用户ID

7. 添加一条会员信息
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

7.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 POST /enterprise/associator-company/add

7.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

 添加一条会员信息

7.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description
=============== =============== =============================================
 associatorid       String                        当前用户id
--------------- --------------- ---------------------------------------------
     coId           String                         公司ID
--------------- --------------- ---------------------------------------------
     type           Number                       0代表会员
--------------- --------------- ---------------------------------------------
     name           String                      员工真实姓名
--------------- --------------- ---------------------------------------------
     dept           String                         部门
--------------- --------------- ---------------------------------------------
     code           String                      员工证件号
--------------- --------------- ---------------------------------------------
     path           String              证件url地址，格式为 文件名:url地址
=============== =============== =============================================

7.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

    {
        "associatorid": 12312321,
        "coId":"13",
        "name":"xiao'ming",
        "dept":"develop",
        "code": "1324",
        "path": "dlas:/adagds.jpg"
    }
