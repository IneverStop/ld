Book模块
===============

端口号
-----------
 gateway入口端口号9999
 原端口20000

接口
--------

1. 获取指定用户的书籍杂志书架
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

1.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 GET /book/shelf

1.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

 调用了ld-book的相应接口

1.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description
=============== =============== =============================================
    userId         String                         用户ID
=============== =============== =============================================

1.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

    GET /book/shelf?userId=100

1.5 Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description
=============== =============== =============================================
    status          String                           状态码
--------------- --------------- ---------------------------------------------
    message         String                          返回消息
--------------- --------------- ---------------------------------------------
     data
--------------- --------------- ---------------------------------------------
   bookShelf                                     书籍书架
--------------- --------------- ---------------------------------------------
      id            Number                        书籍ID
--------------- --------------- ---------------------------------------------
     photo          String                        书籍封面
--------------- --------------- ---------------------------------------------
     name           String                       书籍名
--------------- --------------- ---------------------------------------------
     ispay          Number                       是否要付费
--------------- --------------- ---------------------------------------------
    ispublic        Number                       是否公开
--------------- --------------- ---------------------------------------------
    price           Number                       价格
--------------- --------------- ---------------------------------------------
 magazineShelf                                  杂志书架
--------------- --------------- ---------------------------------------------
      id            Number                       杂志ID
--------------- --------------- ---------------------------------------------
     photo          String                       杂志封面
--------------- --------------- ---------------------------------------------
     name           String                       杂志名
--------------- --------------- ---------------------------------------------
     ispay          Number                       是否要付费
--------------- --------------- ---------------------------------------------
    ispublic        Number                       是否公开
--------------- --------------- ---------------------------------------------
    price           Number                       价格
=============== =============== =============================================
