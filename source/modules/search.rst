Search模块
===============

端口号
-----------
 gateway入口端口号9999
 原端口20110

接口
--------

1. 新增用户搜索记录
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

1.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 Post /search/addSearchHistory

1.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

 新增用户搜索记录

1.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description
=============== =============== =============================================
    searchKey       String                      搜索的关键词
--------------- --------------- ---------------------------------------------
 searchModule       Number      搜索的模块  1 : 表示首页  2：图书 3：杂志 4 课程
--------------- --------------- ---------------------------------------------
 operationUser      Number                  当前用户的id
=============== =============== =============================================

1.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::


    GET /search/addSearchHistory

1.5 Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description
=============== =============== =============================================
    status          String                           状态码
--------------- --------------- ---------------------------------------------
    message         String                          返回消息
--------------- --------------- ---------------------------------------------
     data            NULL
=============== =============== =============================================

1.6 Sample Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 {
    "status": "SEARCH-SUCCESS-101",
    "message": "添加成功",
    "data": null
}

---------------------------------------------


2. 用户搜索记录和热门搜索
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

2.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 Get /search/querySearchHistory/searchModule?associatorid=xx

2.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

  用户搜索记录和热门搜索

2.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description
=============== =============== =============================================
 searchModule       Number      搜索的模块  1 : 表示首页  2：图书 3：杂志 4 课程
--------------- --------------- ---------------------------------------------
 operationUser      Number                  当前用户的id
=============== =============== =============================================

2.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::


    GET /search/addSearchHistory

2.5 Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description
=============== =============== =============================================
    status          String                           状态码
--------------- --------------- ---------------------------------------------
    message         String                          返回消息
--------------- --------------- ---------------------------------------------
     data
--------------- --------------- ---------------------------------------------
   hotSearch                                        热门历史搜索
--------------- --------------- ---------------------------------------------
   searchKey        String                        搜索关键词名称
--------------- --------------- ---------------------------------------------
     nums           Number                         搜索关键词数量
--------------- --------------- ---------------------------------------------
  ownerSearch                                     个人历史搜索
--------------- --------------- ---------------------------------------------
  searchKey        String                        搜索关键词名称
--------------- --------------- ---------------------------------------------
     nums           Number                       搜索关键词数量
=============== =============== =============================================

2.6 Sample Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

   {
        "status": "SEARCH-SUCCESS-104",
        "message": "查询成功",
        "data": {
            "hotSearch": [{
                "searchKey": "艺术",
                "nums": 1
            }],
            "ownerSearch": [{
                "searchKey": "艺术",
                "nums": 1
            }]
        }
    }
