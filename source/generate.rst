4. 搜索结果列表
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

4.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 Post /search/queryListOnSearch

4.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

  搜索结果列表

4.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
searchModule       Number      搜索的模块  1 : 表示首页  2：图书 3：杂志 4 课程
--------------- --------------- ---------------------------------------------
 searchKey        String                        搜索关键词名称
=============== =============== =============================================

4.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::
    Post /search/queryListOnSearch
    {
	    "searchModule" : 3,
	    "searchKey" : "1"
    }
