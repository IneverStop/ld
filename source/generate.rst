5. 按照行业获取所有企业
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

5.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 GET /enterprise/company/all/trade/{tradename}

5.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

 按照行业获取所有企业以及用户是否加入企业的信息

5.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
   tradename        String      
--------------- --------------- ---------------------------------------------
    userId          String      
=============== =============== =============================================

5.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 /enterprise/company/all/trade/文化?userId=1340
