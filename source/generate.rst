7. 添加一条会员信息
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

7.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 POST /enterprise/associator-company/add

7.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

 添加一条会员信息

7.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
 associatorid       String      
--------------- --------------- ---------------------------------------------
     coId           String      
--------------- --------------- ---------------------------------------------
     name           String      
--------------- --------------- ---------------------------------------------
     dept           String      
--------------- --------------- ---------------------------------------------
     code           String      
--------------- --------------- ---------------------------------------------
     path           String      
=============== =============== =============================================

7.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::


    {
        "associatorid": 12312321,
        "coId":"13",
        "name":"xiao'ming",
        "dept":"develop",
        "code": "1324",
        "path": "dlas:/adagds.jpg"
    }

7.5 Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
    status          String                           状态码                     
--------------- --------------- ---------------------------------------------
    message         String                          返回消息                     
--------------- --------------- ---------------------------------------------
     data           Number      
--------------- --------------- ---------------------------------------------
=============== =============== =============================================

7.6 Sample Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::


    {
      "status": "ENTERPRISE-SUCCESS-101",
      "message": "添加成功",
      "data": 229
    }
