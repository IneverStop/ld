3. 用户搜索记录删除
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

3.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 Delete /search/deleteSearchHistory

3.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

  用户搜索记录删除

3.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
 searchModule       Number      搜索的模块  1 : 表示首页  2：图书 3：杂志 4 课程
--------------- --------------- ---------------------------------------------
 operationUser      Number                  当前用户的id
=============== =============== =============================================

3.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 
    Delete /search/deleteSearchHistory

3.5 Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
    status          String                           状态码                     
--------------- --------------- ---------------------------------------------
    message         String                          返回消息                     
--------------- --------------- ---------------------------------------------
     data            NULL
=============== =============== =============================================

3.6 Sample Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

   {
        "status": "SEARCH-SUCCESS-102",
        "message": "删除成功",
        "data": null
    }
