6. 修改密码
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

6.1 访问路径
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 POST /user/password/reset

6.2 逻辑
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

 修改密码

6.3 Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
      id            String      
--------------- --------------- ---------------------------------------------
     phone          String      
--------------- --------------- ---------------------------------------------
    newPSD          String      
=============== =============== =============================================

6.4 Sample Request
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

 {
        "id": 1348,
        "phone": "18136085708",
        "newPSD" : "e10adc3949ba59abbe56e057f20f883e"
    }

6.5 Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
=============== =============== =============================================
  Field Name         Type                        Description                 
=============== =============== =============================================
    status          String                           状态码                     
--------------- --------------- ---------------------------------------------
    message         String                          返回消息                     
--------------- --------------- ---------------------------------------------
     data            NULL       
--------------- --------------- ---------------------------------------------
=============== =============== =============================================

6.6 Sample Response
>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>
::

   {
      "status": "USER-SUCCESS-103",
      "message": "密码成功修改",
      "data": null
    }
